# Walk to Unlock

Приложение на React Native (Expo), которое мотивирует ходить пешком, чтобы зарабатывать экранное время для TikTok, YouTube и других приложений.

## Возможности
- Синхронизация с Apple HealthKit и Google Fit для чтения шагов и расстояния.
- Автоматический расчёт экранного времени на основе пройденного расстояния.
- Магазин с подписками и бустерами (интеграция с Google Play Billing и App Store).
- Геймификация: бейджи, уровни и прогресс.
- Гибкая настройка ограничений приложений через Screen Time API / App Usage API.
- Напоминания о скором окончании экранного времени.

## Старт проекта
```bash
npm install
npm run start
```

### iOS
```bash
npm run ios
```

### Android
```bash
npm run android
```

## Структура
- `App.tsx` — корневой компонент с провайдерами и навигацией.
- `src/screens` — экраны (Домой, Шаги, Магазин, Ачивки, Настройки).
- `src/services` — интеграции со здоровьем, биллингом и ограничением приложений.
- `src/context` — контексты состояния (шаги, подписки, настройки).
- `src/components` — переиспользуемые UI компоненты.
