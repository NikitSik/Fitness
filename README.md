# Walk to Unlock 2.0

Мобильное приложение на Expo / React Native, которое мотивирует гулять, чтобы разблокировать экранное время для TikTok, YouTube и других приложений. Интерфейс выполнен в современном градиентном стиле с карточками и нижней навигацией.

## Основные возможности
- **Домой**: оставшееся экранное время, прогресс дня, акция с множителем и кнопка *Go for a walk*.
- **Шаги**: аналитика за день и неделю, визуализация прогресса и статистика дистанции.
- **Магазин**: тарифы (100/75/50 м за 10 минут), безлимит и x2 бустер с интеграцией в Google Play Billing / App Store.
- **Ачивки**: бейджи за дистанцию, уровни и серия дней.
- **Настройки**: выбор блокируемых приложений и уведомления «осталось 5 минут».
- **Ограничения приложений**: заглушки для Screen Time API (iOS) и App Usage API (Android) с автоматической блокировкой при исчерпании минут.

## Запуск
```bash
npm install
npm run start
```

Для эмуляторов используйте `npm run ios` или `npm run android`.

## Архитектура
- `App.tsx` — навигация и глобальные провайдеры состояния.
- `src/providers/AppProviders.tsx` — компоновка Activity / Subscription / Settings контекстов.
- `src/context` — бизнес-логика: шаги и начисления, подписки, выбор приложений.
- `src/screens` — экраны с готовым UI и логикой.
- `src/components` — градиентный контейнер, карточки, прогресс-бары, графики.
- `src/services` — заглушки интеграций HealthKit/Google Fit, IAP и блокировки.
